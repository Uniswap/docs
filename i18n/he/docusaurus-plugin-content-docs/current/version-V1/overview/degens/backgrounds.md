רקעים של DEGEN הוקצו באקראי עם ההטבעה, מה שהביא לרקע משותף או לאחת משלוש אפשרויות מיוחדות: **נדיר**, **מטא**או **אגדי**. מלבד מראה מגניב, הרקע מגיע עם הטבות מיוחדות כמו מראה מכובד במשחק ומכפילי רווחים נוספים.

### מכפילי רווחים

- נדיר: 1.5
- מטא: 3
- אגדי: 6
